# ServeLink Landing Page

**Site:** servelinkapp.com
**Stack:** Next.js on Vercel, Supabase (waitlist table)
**Admin:** /admin (password: ServeLink@Admin2026)

## Completed Features

- Waitlist form → Supabase database
- Admin dashboard with CSV export
- Trust badges (SSL, no spam, privacy)
- QR codes for waitlist
- Social media footer icons (TikTok, Instagram, Facebook, LinkedIn, Pinterest, Snapchat, Threads)
- Video integration (promo, client tutorial, provider tutorial) in public/videos/
- Watch Demo button → video modal
- Sound effects (subtle, toggleable)
- Bilingual EN/FR
- Privacy Policy + Terms of Service links

## Environment Variables (Vercel)

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `NEXT_PUBLIC_ADMIN_PASS`
